sudo apt update
🔄 Atualiza a lista de pacotes (não instala nada).

sudo apt upgrade
⬆️ Atualiza os pacotes instalados, sem remover nada.

sudo apt full-upgrade
🧠 Atualiza tudo, mesmo que precise remover ou trocar pacotes.

sudo apt dist-upgrade
🧠 Igual ao full-upgrade, nome antigo. Faz upgrades mais “inteligentes”.

sudo apt autoremove
🧹 Remove pacotes que não são mais usados.

sudo apt autoclean
🧼 Apaga arquivos de pacotes antigos, que não são mais instaláveis.

sudo apt clean
🔥 Apaga todos os arquivos baixados de pacotes (.deb), limpa geral.
